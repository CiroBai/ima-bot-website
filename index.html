YouTube 链接 → 字幕提取 → 结构化大纲 → 口播稿 → 配图

上传图片/音频/视频包 → 转码/去噪/提取关键帧/生成字幕/翻译

输出：处理后文件包 + 任务报告
S4

一键发布

已生成内容 → 选择渠道 → 预检 → 确认 → 自动发布

输入：标题 + 正文 + 封面
输出：发布结果 + 失败原因

每日/每周自动监控

监控源 → 定时抓取 → 差异检测 → 摘要推送

输入：URL / 关键词 / 频道
